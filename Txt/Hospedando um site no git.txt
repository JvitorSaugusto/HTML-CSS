Crie um novo repositorio no Git Hub Desktop > vá no site do git > settings > page > Escolha um tema para adicionar o repositorio inteiro.
Quando for somente um site do repositorio não coloque um tema > va em source e main.

Após gerar o link do seu site, vá para QrCode Generator e cole a url gerada pelo Git Hub la, assim será gerado um Qr code para seu site, facilitando o acesso.
